# ジュニアゴルフ場ポータル

関東エリアのジュニア料金対応ゴルフ場を検索・閲覧できるポータルサイトです。

## 機能

- 🔍 ゴルフ場名、地域、タグでの検索
- 🗺️ 都道府県別フィルター
- 📊 ソート機能（料金順、評価順など）
- 📄 各ゴルフ場の詳細情報ページ
- 🔗 公式サイト・楽天ゴルフへのリンク

## ファイル構成

- `index.html` - トップページ（検索・ソート機能付き）
- `course.html` - 個別ゴルフ場詳細ページ
- `styles.css` - スタイルシート
- `script.js` - 検索・ソート・フィルター機能
- `golf-courses.js` - ゴルフ場データ

## 使い方

ブラウザで `index.html` を開いてご利用ください。

## デプロイ

Vercelでデプロイ可能です。
